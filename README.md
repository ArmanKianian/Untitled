# Untitled
 "The Very Serious Juniper Dev Game Jam" Project
 
Tutorial:

Just try to kill next slot monster by spinning wheel and getting units and placing them with mouse. it's just drag & drop. if anyone win the battle, enemy lose one ground. ground can be get back by winning next battle. each card staying in ground after round is a point, whoever have more points wins the battle.(for now pls refresh game for a new round XD)
